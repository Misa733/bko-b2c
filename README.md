# Cockpit Comercial B2C

MVP interno para upload manual de planilhas `.xlsx`, classificacao automatica, consolidacao por competencia e visualizacao gerencial.

## Rodando

```bash
npm install
npm run dev
```

Backend: `http://localhost:3333`

Frontend: `http://localhost:5173`

## Funcionalidades do MVP

- Criacao e exclusao de competencias
- Upload multiplo de planilhas Excel
- Identificacao automatica por colunas
- Consolidacao por CPF ou Nome Cliente + Nome Vendedor
- Dashboard geral
- Telas de vendedores, clientes e relatorios
- Exportacao CSV e Excel
- Persistencia local em JSON
